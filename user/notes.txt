!!!Чтобы команда разработчиков знала о том, какие пакеты для node в проекте были мной установлены,
нужно к каждой команде установки пакетов приписывать команду -save
Папка node_modules по умолчанию и в принципе всегда внесена в .gitignore,
а все пакеты устанавливаются в нее. Добавленная команда -save к основной команде установки пакетов
внесет информацию об установленных пакетах в файл package-lock.json , таким образом команда разработчиков
сможет иметь доступ к этой информации.
А также информация появится в файле package.json об установленных пакетах. Другой разработчик из команды
получив из github файл package.json сможет установить себе на локальный компьютер в проект недостающие
у него пакеты node_modules , запустив в терминале из корневого каталога проекта команду
npm i
пример полной команды для командной разработки:
npm i react-router-dom -save
